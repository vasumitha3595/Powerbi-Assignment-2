## 📌 Overview

This dashboard provides an executive view of business performance using
KPIs and interactive visualizations. It enables users to monitor sales
trends, compare actual sales against targets, analyze profit margins,
and explore geographic performance.

## ✨ Features

-   KPI Cards (Average Profit, Total Orders, YTD Sales)
-   Sales Target Achievement by Category
-   Profit Margin by Sub-Category
-   Monthly Sales Trend
-   Sales Performance Matrix
-   Quantity vs Profit Analysis
-   Geographic Sales Map
-   State-wise Order Analysis
-   Sales Distribution Treemap
-   Interactive City Slicer
-   Multi-page navigation

## 🛠️ Technologies

-   Microsoft Power BI
-   Power Query
-   DAX
-   Excel

## ▶️ How to Use

1.  Download the `.pbix` file.
2.  Open it with Microsoft Power BI Desktop.
3.  Use the City slicer to filter the report.
4.  Navigate between Page 1 and Page 2.

-   Monitor Year-to-Date Sales.
-   Compare Actual Sales vs Target.
-   Identify top-performing categories and sub-categories.
-   Analyze monthly sales trends.
-   Explore state and city level sales performance.
